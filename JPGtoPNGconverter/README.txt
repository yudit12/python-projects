JPGtoPNGconverter

what the programe do? 
changing format of all images in given folder
from jpg to png  and save the images.png in new folder

run program  :
 *from command line : ./JPGtoPNGconverter.py pokedex\ pngImages\
 or 
*using pycharm :
for runing program with arguments
- goto  Run
- edit configuration
- set argument in the parameters field  ( for example pokedex\ pngImages\) - two parameters of folders in windows
- click Run
